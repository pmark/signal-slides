# Custom AI Setup & Memory

## Language and Terminology Guidelines
- **Use Straightforward Language:** Always use plain, straightforward terms and language in both the frontend text and when communicating with the user.
- **Avoid Jargon & Pretentious Framing:** Do NOT use tech jargon, overly academic language, or "pretentious baloney" (e.g., use "Slide Notes" instead of "Narrative Synthesis", "Key Claims" instead of "Atomic Deconstruction", "Clear statements" instead of "Atomicity").
- **Prioritize Clarity:** If an idea can be expressed with a standard, commonly understood term, prefer it over an invented or overly complex term.
